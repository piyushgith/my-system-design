# Behavioral Interview Stories

STAR format: Situation → Task → Action → Result

## Suggested story banks

```
behavioral/
  leadership-ownership.md     — drove initiative, took accountability
  conflict-resolution.md      — disagreed with team / manager
  technical-influence.md      — changed architectural direction
  failure-learning.md         — production incident, missed deadline
  cross-team-collaboration.md — aligned stakeholders, unblocked teams
```

## Template per story

```
## Story: {Short title}

**Question it answers:** "Tell me about a time you..."

**Situation:** (2 sentences — context, stakes)

**Task:** (1 sentence — your specific responsibility)

**Action:** (3-4 bullets — what YOU specifically did)

**Result:** (metric + qualitative impact)

**Follow-up prep:**
- What would you do differently?
- How did it affect the team long-term?
```
